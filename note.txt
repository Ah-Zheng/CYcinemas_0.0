# 資料庫

    會員
    電影
    訂單

# 功能

    1. 會員
        => 註冊
        => 登入
        => 集點

    2. 電影資訊
        => 爬蟲爬資料 ( 片名、語言、上映日期、時長、播映時刻、電影簡介 )
        => 電影時刻
        => 分級

    3. 訂票
        => 價格
        => 套票 ( 附餐 )
        => 劃位 ( 交易機制 )
        => 折扣 ( 生日 )
        => 結帳
    
    4. 抽獎

    5. 報表 ( 周報表 )
